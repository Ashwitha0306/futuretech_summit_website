# 🚀 FutureTech Summit 2026

A modern, futuristic, and fully responsive conference landing page designed for **FutureTech Summit 2026**. This project showcases a professional technology event website featuring speakers, schedules, ticket pricing, AI-powered FAQ assistance, registration forms, and interactive user experience.

## 🚀 How to Run

### Method 1: Open Directly

1. Download or clone the repository.
2. Navigate to the project folder.
3. Open `index.html` in your browser.

### Method 2: Using VS Code Live Server

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
